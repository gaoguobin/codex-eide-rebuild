# Code of Conduct

## Our pledge

We pledge to make participation in this project a respectful and constructive experience for everyone.

## Our standards

Examples of behavior that supports a positive environment include:

- Using welcoming and inclusive language
- Respecting differing viewpoints and experiences
- Giving and accepting constructive feedback
- Focusing on what is best for the project and community

Examples of unacceptable behavior include:

- Harassment, intimidation, or discrimination
- Personal attacks or derogatory comments
- Publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.
